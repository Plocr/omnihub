# Mintlify Design System

## Visual Theme
Clean, documentation-first design with excellent readability. White backgrounds, green accent, clear typographic hierarchy.

## Color Palette
- Background: `#ffffff` | `#f8fafc` (secondary bg)
- Text primary: `#0f172a`
- Text muted: `#64748b`
- Accent/Primary: `#0d937d` (emerald green)
- Border: `#e2e8f0`
- Error: `#ef4444`

## Typography
- Font: Inter (sans-serif)
- Monospace: JetBrains Mono
- Sizes: 14px body, 12px small, 24px h2, 30px h1

## Components
- Cards: White, 1px border, shadow-sm, rounded-lg
- Buttons: Solid green (`#0d937d`) or outline, rounded-lg
- Badges: Small, colored backgrounds, rounded-full
- Inputs: White, 1px border, focus ring green

## Layout
- Max-width content: 1280px
- Sidebar: 260px on desktop, hidden on mobile
- Spacing: 4px base grid, 16px/24px/32px/48px/64px scale
